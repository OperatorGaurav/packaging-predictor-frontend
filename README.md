## 🚀 Setup Guide (Step by Step)

### Step 1 — Install dependencies
Open terminal inside this folder and run:
```
npm install
```

---

### Step 2 — Get your FREE OpenRouter API Key

1. Go to: https://openrouter.ai
2. Click "Sign Up" — use Google or GitHub (no credit card needed)
3. After signing in, go to: https://openrouter.ai/keys
4. Click "Create Key"
5. Give it a name like "packaging-app"
6. Copy the key — it starts with "sk-or-..."

---

### Step 3 — Create your .env file

In your terminal run:
```
copy .env.example .env
```

Then open the .env file in Notepad and replace the placeholder:
```
VITE_OPENROUTER_API_KEY=sk-or-xxxxxxxxxxxxxxxxx
```

---

### Step 4 — Run the app
```
npm run dev
```

Open your browser at: http://localhost:5173
